# React Task Manager (To-Do App)

## Experiment 7 – React UI

### 📌 Project Overview

This project is a simple **Task Manager (To-Do Application)** built using **React**.
The application allows users to add tasks, mark tasks as completed, delete tasks, and dynamically view their task list.

The goal of this experiment is to understand **React fundamentals such as state management, event handling, and list rendering**.

---

## 🎯 Objectives

* Understand **React state management using useState**
* Learn **event handling in React**
* Practice **list rendering using map()**
* Implement **basic CRUD operations**
* Build a **real-world interactive UI application**

---

## ⚙️ Technologies Used

* React
* JavaScript
* Vite
* HTML5
* CSS3

---

## 🧩 React Concepts Implemented

### 1. State Management

The application uses the **useState hook** to store and manage tasks.

### 2. Event Handling

User actions such as:

* Adding a task
* Marking a task as completed
* Deleting a task

are handled through React event handlers.

### 3. List Rendering

Tasks are displayed dynamically using the **map() function**.

### 4. Conditional Rendering

* If no tasks exist → a message **"No tasks available"** is displayed.
* Completed tasks are displayed with **strikethrough styling**.

## 🧑‍💻 Application Features

The application includes the following features:

* Add a new task

* Delete tasks
